## 1. 使用说明

本文件将 Excel 模板中的“字段填写要求/批注要求”转为可执行规则，分为三层：
- 字段级规则：单字段可校验规则
- 跨表级规则：两个及以上表之间一致性规则
- 流程级规则：状态流转与先后约束规则

规则字段定义：
- `规则ID`：唯一标识，格式 `F-`/`C-`/`P-`
- `来源`：模板正文 / 单元格批注 / 业务约定
- `触发点`：录入提交 / Excel 导入预览 / Excel 入库 / 编辑保存
- `校验逻辑`：可开发实现描述
- `错误提示`：建议返回给前端的提示语
- `优先级`：P0（必须）/ P1（应实现）/ P2（优化）
- `状态`：已实现 / 部分实现 / 未实现

## 5. 落地建议（开发顺序）

1. 第一阶段（P0 必须规则）
- 先完成 P0 的字段级 + 跨表级 + 流程级校验。
- 优先落在后端（接口层 + 导入层）统一校验，前端做即时提示。

2. 第二阶段（P1 规则）
- 加入文本质量类规则（“事实描述”“禁止空泛语句”等）。
- 加入流程联动规则（10 批准后强制同步任务）。

3. 第三阶段（P2 规则）
- 加入文案规范建议、智能提示、自动修复建议。

## 6. 当前系统实现覆盖总结

- 已实现：
  - 基础枚举校验（阶段/状态/信号灯/是否）
  - 年度与进度数值范围校验
  - 导入清洗、项目复用、部分去重
  - 角色 + 项目权限双重写入控制
- 部分实现：

- 未实现：
  - 大部分跨表一致性规则
  - 多数流程级强制口径
  - 批注中的文本质量与管理口径规则
